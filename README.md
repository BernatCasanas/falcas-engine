## Controls
* WASD: Camera Movement
* Left Click + Drag: Camera Rotation
* Mouse Wheel || Alt + Left Click + Drag: Camera Zoom
* F: Object Focus
* Shift + WASD: x2 Speed
* Alt + Right Click: Orbitate Selected Object

## Funcionality 
* Drag and Drop:
** FBX
** PNG
** DDS
* Create Primitives in the upside menu
